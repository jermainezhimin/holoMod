holoModDemoCompressed.mp4 is a demo video of our project, please watch it for a better understanding on the features of our software.

Build.zip contains the executable file of our software for both Windows & Mac.

HolModBasic.zip contains the Unity source code to build our software.

printPrism.m is the Matlab script to generate laser cut files for n-sided prism.

Thing.obj is a model which you can use to test the import primitive feature of the software.
